# 将来の実装アイデア

## 方針・目的
- 語群の管理と再利用をしやすくする
- 設定の柔軟性と作業効率の両立

## 追加したい機能
- ディフォルトプロンプト（高画質など）を設定できるようにする
- プロンプトの入れ替え機能（生成済みプロンプト同士の並び替え）
- 固定品質プロンプトの実装（常に一定の品質関連語を付与）
- プリセット切り替え（用途別の設定セット）
- お気に入りはJSONに書き出し/読み込みできるようにする　
- 語群のお気に入り登録（上位固定表示）　OK

## UI/UX 改善
- お気に入りの一覧・検索・タグ付け
- フォルダ/語句のドラッグ移動、複数選択、一括削除/移動

## データ/保存/共有
- フォルダ・ワードの整理の仕様の改善
- 同名のお気に入りは不可（重複名を禁止）
- お気に入りリストは並べ替え可能（あいうえお順/作成日順など）
- 語群データの部分インポート/追加（特定カテゴリのみ取り込み、既存データにマージ）
読み込み時に上書き読み込み（読み込む語群に登録されていない語群は消える）もしくは
追加登録（今の語群にない語群だけ読み込み）の選択をできるようにする
